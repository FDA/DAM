Spreadsheets for the Daily Accumulation Model (DAM) detailed in https://www.mdpi.com/2305-6304/13/8/649

SupplementaryMaterial.xlsx contains the graphs and sheets discussed in the paper. Monte Carlo modeling and analysis is through the commercial @Risk software.

DAM_Scenario1.xlsm is a macro enabled spreadsheet for scenario 1 in the paper. @Risk is no longer required due to the inclusion of macros.
